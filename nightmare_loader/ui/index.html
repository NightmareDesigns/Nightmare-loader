Easy fix: run nightmare-loader install-launcher once to create a desktop shortcut, then launch Nightmare Loader from your application menu — it will ask for your password automatically. Alternatively: sudo nightmare-loader ui
🪟 WINDOWS MODE — ISO inspection and USB drive detection are supported. Partitioning + GRUB installation require Linux (or WSL 2). To prepare a drive: install Nightmare Loader inside WSL 2, then run sudo nightmare-loader prepare /dev/sd<x> from the WSL terminal.

⚙️ Prepare Drive

⚠ WARNING: All data on the selected device will be erased. GRUB2 will be installed for both UEFI and legacy BIOS.

🖥 BOOT FROM USB – QUICK GUIDE
  1. Insert the prepared USB drive and restart your computer.
  2. Enter your BIOS / UEFI firmware (typically F2, F12, Del or Esc during POST).
  3. Set the boot order so the USB drive is first, or choose it from the one-time boot menu.
  4. Save and exit — the Nightmare Loader GRUB menu will appear.
  5. Select an ISO entry to boot it.

UEFI Secure Boot: may need to be disabled if your ISOs are unsigned. Look for "Secure Boot" in your UEFI security settings and set it to Disabled.
