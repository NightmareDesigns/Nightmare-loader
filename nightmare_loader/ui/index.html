⚠ WARNING: All data on the selected device will be erased. GRUB2 will be installed for both UEFI and legacy BIOS.

🖥 BOOT FROM USB – QUICK GUIDE
  1. Insert drive and restart.
  2. Enter BIOS / UEFI (F2 F12 Del).
  3. Set USB as first boot device.
  4. Save & exit — Nightmare Loader GRUB menu appears.
  5. Select an ISO or OS to boot.

UEFI Secure Boot may need to be disabled for unsigned ISOs.
